## Project Overview
This repository contains the source code for the **Eid Reward Distribution System (ERDS)**. It is designed to verify family member identities and securely sync data packets to unlock digital rewards (Eidi).

## Features
* **Biometric Authentication**: Utilizes local camera feeds for real-time recipient verification.
* **Integrity Sync**: Multi-stage data packet synchronization (7-stage protocol).
* **Fault Tolerance**: Automatic recovery protocols for "Unexpected Entity" detection.
* **Database**: Local mapping for high-priority targets.

## Deployment
1. Ensure all `JPEG/PNG` assets are in the root directory.
2. Launch `INDEX.html` via a secure browser.
3. Complete the 7-stage sync to unlock the vault.
